# Logo de la Institución

## Cómo personalizar el logo

Para personalizar el logo de la institución en el menú lateral:

1. Coloca tu logo en esta carpeta con el nombre `logo.png`
2. El logo debe ser cuadrado (preferiblemente 128x128 píxeles o similar)
3. Formato recomendado: PNG con fondo transparente

## Ubicación del archivo

```
assets/img/institution/logo.png
```

## Comportamiento

- Si existe `logo.png` en esta carpeta, se mostrará en el menú lateral
- Si no existe, se mostrará el logo por defecto de EgresApp2
- El logo se muestra en formato circular automáticamente

## Para implementar en otro establecimiento

Simplemente reemplaza el archivo `logo.png` en esta carpeta con el logo de la nueva institución.
